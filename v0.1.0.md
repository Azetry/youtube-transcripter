## Summary

Initial release of YouTube Transcripter, a complete solution for transcribing YouTube videos using OpenAI Whisper API with automatic GPT-powered text correction. Includes CLI, REST API, and Vue 3 web interface with Docker deployment support.

## Changes

### FEAT
- Add YouTube video information extraction module using yt-dlp
- Add Whisper API integration for speech-to-text transcription
- Add GPT-powered text correction with context-aware processing
- Add visual diff viewer for comparing original and corrected transcripts
- Add FastAPI backend with async task processing and SSE progress updates
- Add Vue 3 frontend with real-time progress indication and transcript display
- Add CLI interface with interactive mode support

### ADD
- Add project directory structure with downloads and transcripts folders

- Add MIT LICENSE file
